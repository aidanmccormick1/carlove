# Future Car Love Story Game

A lightweight browser game that helps an indecisive buyer choose a specific used car match (year, mileage, price), with AI-style compatibility scoring and decision guidance.

The quiz now runs as full-screen, mobile-optimized steps with fun extra questions for better matching.

## Run it

1. Open `/Users/aidanmccormick/Documents/New project/index.html` in any modern browser.
2. Fill out the car "dating" quiz.
3. Click **Find my car soulmate** to see:
   - Exact car match (year, model, mileage, price)
   - Why it fits
   - What to watch out for
   - Inspection checklist
   - Top 3 alternatives
